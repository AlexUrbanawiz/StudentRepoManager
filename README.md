# StudentRepoManager
This is a simple PowerShell script and batch file that will help the user manage many repositories. 
I built it to help me manage the repositories for student for my TA job.

To use it, simply run the RunManageStudents.bat file, and follow the prompts given by the powershell window. The students will be stored in the StudentRepos folder that will be created when first run. When you select a student by the given index it will pull their repository and open it in VS Code.

When giving the names, make sure the name has a first and last name, as the names are sorted by the last.

It will not function properly with only 1 repo its managing, so make sure to have at least 2.
